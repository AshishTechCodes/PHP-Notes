PHP String Operations

Original String

<php $x = "Ashish Gupta"; echo "\$x = \"$x\""; >
String Output

Displaying the string variable:

<php echo "Hello $x"; >
String Length

Counts the number of characters in the string:

<php echo strlen($x); > characters
Word Count

Counts the number of words in the string:

<php echo str_word_count($x); > words
Case Conversion

Convert to uppercase and lowercase:

Uppercase: <php echo strtoupper($x); >
Lowercase: <php echo strtolower($x); >
String Replacement

Replace "Ashish" with "Jhon":

<php echo str_replace("Ashish", "Jhon", $x); >
String Reverse

Reverse the characters in the string:

<php echo strrev($x); >
String Position

Find position of "Gupta" in the string:

Position: <php echo strpos($x, "Gupta"); >
String Explode

Split string into array using "h" as delimiter:

<php $y = explode("h", $x); print_r($y); >
String Concatenation

Combine multiple strings together:

<php $hello = "Hello"; $word = "$hello $x"; echo $word; >
String Slicing

Extract parts of the string:

Slice from position 4 (6 chars): <php echo substr($x,4, 6); >
Slice to end from position 4: <php echo substr($x,4); >
Slice from end (-5, 3 chars): <php echo substr($x,-5,3); >
Escape Characters

Include special characters in strings:

<php echo "We are the so-called \"Vikings\" from the north."; >
Current Time

Display current time using date() function:

<php echo date("H:i:s"); >
