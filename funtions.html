Types of Functions in PHP

1. Call by Value

Original value is not modified when passed to function.

function increment($num) {
    $num++;
    return $num;
}
$value = 5;
increment($value);
// Original $value remains unchanged
Output:
Original value: 5
Value after increment inside function: 6
Value after function call: 5

2. Call by Reference

Original value can be modified when passed to function using &.

function incrementRef(&$num) {
    $num++;
}
$value = 5;
incrementRef($value);
// Original $value is modified
Output:
Original value: 5
Value after function call: 6

3. Default Arguments

Parameters can have default values if not provided.

function greet($name = "Guest") {
    return "Hello, $name!";
}
greet();      // Uses default
greet("John"); // Uses provided value
Output:
greet(): Hello, Guest!
greet("John"): Hello, John!

4. Parameterized Function

Functions that accept parameters/arguments.

function add($a, $b) {
    return $a + $b;
}
$result = add(3, 4);
Output:
Result of add(3, 4): 7

5. Returning Values

Functions can return values using return statement.

function multiply($x, $y) {
    return $x * $y;
}
$product = multiply(5, 6);
Output:
Product of 5 and 6: 30

6. Variable-length Arguments

Functions that accept variable number of arguments.

function sum(...$numbers) {
    $total = 0;
    foreach($numbers as $num) {
        $total += $num;
    }
    return $total;
}
$total = sum(1, 2, 3, 4);
Output:
Sum of 1, 2, 3, 4: 10

7. Recursive Functions

Functions that call themselves.

function factorial($n) {
    if ($n <= 1) {
        return 1;
    }
    return $n * factorial($n - 1);
}
$fact = factorial(5);
Output:
Factorial of 5: 120
