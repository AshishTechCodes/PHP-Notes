PHP Arrays Guide

1. Basic Array

Creating and accessing a simple indexed array

// Create array
$fruits = array("Apple", "Banana", "Orange");

// Access elements
echo $fruits[0]; // Apple
echo $fruits[1]; // Banana
echo $fruits[2]; // Orange
Output:
Apple
Banana
Orange

2. Indexed Array

Creating and working with indexed arrays

// Create array (short syntax)
$colors = ["Red", "Green", "Blue"];

// Add element
$colors[] = "Yellow";

// Count elements
echo count($colors);

// Print all elements
print_r($colors);
Output:
4
Array
(
[0] => Red
[1] => Green
[2] => Blue
[3] => Yellow
)

3. Accessing Array Elements

Different ways to access array elements

$numbers = [10, 20, 30, 40, 50];

// Access by index
echo $numbers[2]; // 30

// Access last element
echo $numbers[count($numbers)-1]; // 50

// Access with negative index (PHP 7.1+)
echo $numbers[-1]; // 50
Output:
30
50
50

4. Associative Array

Arrays with named keys (key-value pairs)

// Create associative array
$person = [
"name" => "John Doe",
"age" => 30,
"email" => "john@example.com"
];

// Access elements
echo $person["name"];
echo $person["age"];

// Add new element
$person["city"] = "New York";

print_r($person);
Output:
John Doe
30
Array
(
[name] => John Doe
[age] => 30
[email] => john@example.com
[city] => New York
)

5. Creating Arrays

Different ways to create arrays in PHP

// 1. Using array()
$arr1 = array(1, 2, 3);

// 2. Using short syntax []
$arr2 = [1, 2, 3];

// 3. Range function
$numbers = range(1, 5);

// 4. Array with mixed types
$mixed = [1, "two", 3.0, true];

print_r($numbers);
print_r($mixed);
Output:
Array
(
[0] => 1
[1] => 2
[2] => 3
[3] => 4
[4] => 5
)

Array
(
[0] => 1
[1] => two
[2] => 3
[3] => 1
)

6. Array Operations

Common array operations and functions

$array1 = ["a", "b", "c"];
$array2 = ["d", "e", "f"];

// Merge arrays
$merged = array_merge($array1, $array2);

// Slice array
$slice = array_slice($merged, 2, 3);

// Search in array
$key = array_search("e", $merged);

// Sort array
sort($merged);

print_r($merged);
print_r($slice);
echo "Key of 'e': $key";
Output:
Array
(
[0] => a
[1] => b
[2] => c
[3] => d
[4] => e
[5] => f
)

Array
(
[0] => c
[1] => d
[2] => e
)

Key of 'e': 4

7. Multidimensional Arrays

Arrays containing other arrays

$students = [
["name" => "John", "grade" => "A"],
["name" => "Sarah", "grade" => "B"],
["name" => "Mike", "grade" => "A-"]
];

// Access element
echo $students[1]["name"]; // Sarah

// Add new student
$students[] = ["name" => "Emma", "grade" => "B+"];

print_r($students);
Output:
Sarah
