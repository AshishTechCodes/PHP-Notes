PHP Form Handling

Form Submission Example

This form submits data to itself using POST method

<form action="<?php echo htmlspecialchars($_SERVER["PHP_SELF"]);?>" method="POST">
Name: <input type="text" name="name"><br>
E-mail: <input type="text" name="email"><br>
<input type="submit">
</form>

Form Output Example

How the submitted data would be displayed:

Welcome <?php echo $_POST["name"]; ?><br>
Your email address is: <?php echo $_POST["email"]; ?>

Output Preview:

Welcome John Doe

Your email address is: john@example.com

Note: This is a static HTML representation. In a real PHP environment, the output would dynamically show whatever values were submitted through the form.

Security Considerations:

  • The htmlspecialchars() function prevents XSS attacks when echoing user input
  • For production use, always validate and sanitize user input
  • Consider using prepared statements if storing data in a database
