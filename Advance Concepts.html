PHP Concepts Guide

1. Include & Require

Including external PHP files in your script

// header.php
<?php
$title = "My Website";
?>

// index.php
<?php
include 'header.php';
require 'config.php'; // Stops execution if file not found
echo "Welcome to $title";
?>
Output:
Welcome to My Website
Note: include continues execution if file not found, require stops execution.

2. Filtering

Validating and sanitizing data with PHP filters

$email = "user@example.com";
$url = "https://example.com";

// Validate email
if (filter_var($email, FILTER_VALIDATE_EMAIL)) {
echo "Valid email";
}

// Sanitize string
$dirty = "<script>alert('xss')</script>";
$clean = filter_var($dirty, FILTER_SANITIZE_STRING);
echo $clean;
Output:
Valid email
alert('xss')

3. Sessions

Storing user data across multiple pages

// page1.php
<?php
session_start();
$_SESSION['username'] = "john_doe";
$_SESSION['last_login'] = date('Y-m-d H:i:s');
?>

// page2.php
<?php
session_start();
echo "Welcome back " . $_SESSION['username'];
echo "Last login: " . $_SESSION['last_login'];
?>
Output:
Welcome back john_doe
Last login: 2023-06-15 14:30:22

4. Cookies

Storing small amounts of data on the client side

// Set cookie
setcookie("user", "john_doe", time() + (86400 * 30), "/");

// Access cookie
if (isset($_COOKIE['user'])) {
echo "Welcome " . $_COOKIE['user'];
}

// Delete cookie
setcookie("user", "", time() - 3600, "/");
Output:
Welcome john_doe

5. Callback Functions

Passing functions as arguments to other functions

function greet($name, $callback) {
$greeting = "Hello, $name!";
$callback($greeting);
}

// Define callback
function display($message) {
echo $message;
}

// Use callback
greet("John", "display");

// Anonymous callback
greet("Sarah", function($msg) {
echo strtoupper($msg);
});
Output:
Hello, John!
HELLO, SARAH!

6. File Handling

Reading from and writing to files

// Write to file
$file = fopen("test.txt", "w");
fwrite($file, "Hello World!");
fclose($file);

// Read from file
$content = file_get_contents("test.txt");
echo $content;

// Append to file
file_put_contents("test.txt", "\nNew line", FILE_APPEND);

// Read line by line
$lines = file("test.txt");
print_r($lines);
Output:
Hello World!

Array
(
[0] => Hello World!
[1] => New line
)

7. Error Handling

Handling errors and exceptions

// Basic error handling
function divide($a, $b) {
if ($b == 0) {
trigger_error("Cannot divide by zero", E_USER_ERROR);
}
return $a / $b;
}

// Try-catch block
try {
$result = divide(10, 0);
echo $result;
} catch (Exception $e) {
echo "Error: " . $e->getMessage();
}

// Custom error handler
set_error_handler(function($errno, $errstr) {
echo "Custom handler: $errstr";
});
echo 10 / 0;
Output:
Error: Cannot divide by zero
Custom handler: Division by zero
