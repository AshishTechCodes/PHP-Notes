PHP Variable Scope Demonstration

1. Global Scope

Code:

$x = 5;
function global_scope(){
    echo "Global Variable x value inside function is $x";
}
global_scope();
echo "Global Variable x value outside function is $x";
?>

2. Local Scope

Code:
<?php
function Local_scope(){
    $y = 10;
    echo "Local Variable y value inside function is $y";
}
Local_scope();
echo "Local Variable y value outside function is $y";
?>
Output:
Local Variable y value inside function is $y

"; } Local_scope_demo(); echo "

Local Variable y value outside function is $y (Notice: Undefined variable)

"; ?>
Explanation: The variable $y is only accessible inside the function where it was declared. It cannot be accessed outside the function, resulting in an undefined variable notice.

3. Using global Keyword

Code:
<?php
$x = 5;
$y = 10;
echo $x." + ".$y." = ";
function global_keyword(){
    global $x,$y;
    $y = $x+$y;
    // Alternative: $GLOBALS['y'] = $GLOBALS['x'] + $GLOBALS['y'];
}
global_keyword();
